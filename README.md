# AbleQuest Fun Learning for Every Ability
- AbleQuest is a professional-grade educational gaming application designed specifically for persons with disabilities (PwD). 
- It combines 8 engaging games with student progress tracking, teacher analytics dashboard, and accessibility-focused design to support daily living skills, cognitive development, and social interaction.
# Features
| Category      | Games                            | Skills Developed                         |
| ------------- | -------------------------------- | ---------------------------------------- |
| Daily Living  | Hygiene Trainer, Time Teller | Handwashing sequence Analog clock reading |
| Social Skills | Emotions, Greetings| Emotion recognitionSocial phrases|
| Cognitive     | Colors,Phonics| Color matchingLetter-sound association|
| Fun           | Tic Tac Toe| Strategic thinking |

Student Flow: Home → Student Portal → Dashboard → Game → Score Saved!
Teacher Flow: Home → Teacher Portal → "admin" → Progress Logs
